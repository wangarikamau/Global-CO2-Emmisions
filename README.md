# Global CO₂ Emissions Dashboard (Tableau)

##  Project Overview
This project analyzes global CO₂ emissions across countries and regions over time to identify emission trends, major contributors, and regional disparities.

The interactive dashboard was built using Tableau and focuses on:
- Emissions trends by country
- Regional comparisons
- Temporal changes in global CO₂ output

##  Live Dashboard
 View the interactive Tableau dashboard here:  
https://public.tableau.com/app/profile/wangari.kamau/viz/GlobalCO2Emmisions_17719636400710/Dashboard1

##  Key Insights
- A small number of countries contribute a disproportionately large share of global emissions.
- Emissions trends vary significantly by region, with some regions stabilizing while others continue to rise.
- Historical data show a clear acceleration in emissions post-industrialization.
